28  One-dimensional Interpolation Functions:-
29  Functions to interpolate the primitive of a function at interfaces from its
30  cell-centered values.
31 
32  Arguments:-
33 
34  fI double* array of size (N+1) in the interpolation direction; will contain
35  the interpolated interface function primitive
36  (**needs to be allocated by the calling function)
37  (**does not have ghost points))
38 
39  fC double* array of size (N+2*ghosts) in the interpolation direction of the
40  cell centered function values
41  (**does have ghost points))
42 
43  u double* used only by the characteristic-based interpolation schemes
44  array of size (N+2*ghosts) in the interpolation direction of the
45  cell centered conserved variable (needed to calculate the eigen-
46  decomposition at the interfaces)
47 
48  x double* grid point locations along the 1D line on which the interpolation
49  is being carried out
50  array of size (N+2*ghosts)
51  Used only by non-uniform-grid interpolation schemes
52 
53  upw int upwind direction for non-central schemes
54  (1 -> left biased, -1 -> right biased)
55 
56  dir int direction/dimension along which to carry out interpolation for
57  multi-dimensional domains
58  (eg: 0 for 1D; 0 or 1 for 2D; 0,1 or 2 for 3D)
59 
60  s void* object of type depending on the main solver, should have at least
61  the following data:
62  + ghosts : number of ghosts points in the domain
63  + ndims : number of dimensions
64  + nvars : number of variables per grid point
65  (i.e. size of vector variable being interpolated)
66  + dim_local : local (of this process) dimensions of the domain
67  (integer array of size ndims, with the number of
68  points in each dimension as elements)
69  m void* object containing MPI domain decomposition related variables
70  (this information is used only by compact interpolation schemes)
71 
72  uflag int flag to indicate where the flux function or the solution function
73  is being interpolated
74  (1 -> u; 0 -> f(u) )
75 
76  Notes:
77  + arrangement of points along the direction of interpolation is
78 
79  |<--ghosts-->|0 1 2 ..(interior).. N-1|<--ghosts-->|
80 
81  + number of interfaces are one more than number of interior points
82  + interface i lies between grid points i-1+ghosts and i+ghosts
83 
84  0 1 2 ..... .... N-3 N-2 N-1
85  | x | x | x | x | x | x | x | x | x | x | x | x | x |
86  0 1 2 ... ... ... N-2 N-1 N
87 
88  Returns 0 on normal execution, non-zero on error.

230  int idx; \
231  /* calculate smoothness indicators and the WENO weights */\
232  double b1, b2, b3, a1, a2, a3, a_sum_inv; \
233  for (idx=0; idx<N; idx++) { \
234  b1 = thirteen_by_twelve*(m3[idx]-2*m2[idx]+m1[idx])*(m3[idx]-2*m2[idx]+m1[idx]) \
235  + one_fourth*(m3[idx]-4*m2[idx]+3*m1[idx])*(m3[idx]-4*m2[idx]+3*m1[idx]); \
236  a1 = c1 / ( (b1+weno->eps) * (b1+weno->eps) ); \

238  + one_fourth*(m2[idx]-p1[idx])*(m2[idx]-p1[idx]); \
239  a2 = c2 / ( (b2+weno->eps) * (b2+weno->eps) ); \
