35  One-dimensional Interpolation Functions:-
36  Functions to interpolate the primitive of a function at interfaces from its
37  cell-centered values.
38 
39  Arguments:-
40 
41  fI double* array of size (N+1) in the interpolation direction; will contain
42  the interpolated interface function primitive
43  (**needs to be allocated by the calling function)
44  (**does not have ghost points))
45 
46  fC double* array of size (N+2*ghosts) in the interpolation direction of the
47  cell centered function values
48  (**does have ghost points))
49 
50  u double* used only by the characteristic-based interpolation schemes
51  array of size (N+2*ghosts) in the interpolation direction of the
52  cell centered conserved variable (needed to calculate the eigen-
53  decomposition at the interfaces)
54 
55  x double* grid point locations along the 1D line on which the interpolation
56  is being carried out
57  array of size (N+2*ghosts)
58  Used only by non-uniform-grid interpolation schemes
59 
60  upw int upwind direction for non-central schemes
61  (1 -> left biased, -1 -> right biased)
62 
63  dir int direction/dimension along which to carry out interpolation for
64  multi-dimensional domains
65  (eg: 0 for 1D; 0 or 1 for 2D; 0,1 or 2 for 3D)
66 
67  s void* object of type depending on the main solver, should have at least
68  the following data:
69  + ghosts : number of ghosts points in the domain
70  + ndims : number of dimensions
71  + nvars : number of variables per grid point
72  (i.e. size of vector variable being interpolated)
73  + dim_local : local (of this process) dimensions of the domain
74  (integer array of size ndims, with the number of
75  points in each dimension as elements)
76  m void* object containing MPI domain decomposition related variables
77  (this information is used only by compact interpolation schemes)
78 
79  uflag int flag to indicate where the flux function or the solution function
80  is being interpolated
81  (1 -> u; 0 -> f(u) )
82 
83  Notes:
84  + arrangement of points along the direction of interpolation is
85 
86  |<--ghosts-->|0 1 2 ..(interior).. N-1|<--ghosts-->|
87 
88  + number of interfaces are one more than number of interior points
89  + interface i lies between grid points i-1+ghosts and i+ghosts
90 
91  0 1 2 ..... .... N-3 N-2 N-1
92  | x | x | x | x | x | x | x | x | x | x | x | x | x |
93  0 1 2 ... ... ... N-2 N-1 N
94 
95  Returns 0 on normal execution, non-zero on error.
