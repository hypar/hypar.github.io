Detailed Description

Structure containing all solver-specific variables and functions.

This structure contains all the variables and function pointers for the main PDE solver.

Definition at line 275 of file hypar.h.

int(* JFunction) (double *, double *, void *, int, int)

Pointer to the function to calculate the flux Jacobian for a given solution state (assigned in the physical model initialization called from InitializePhysics()). The flux Jacobian is the Jacobian of the analytical (not spatially discretized) flux for a given solution state (for example, at a grid point). The size is (HyPar::nvars)^2 and the matrix is stored as a 1D array in row-major format.
