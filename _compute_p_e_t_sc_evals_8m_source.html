1 % This script computes the eigenvalues of the linearized
2 % matrix representations of the right-hand-side functions,
3 % when PETSc time integration is used.
4 %
5 % The code needs to be compiled with the "-Dcompute_rhs_operators"
6 % flag so that it writes out files that contain the matrices
7 % in sparse format during a simulation.
8 %
9 % The eigenvalues are written out to text files and can be plotted
10 % in MATLAB with the script PlotPETScEvals.m (or any other utility).
